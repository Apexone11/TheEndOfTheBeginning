# Keyboard Shortcuts Quick Reference

**The End The Beginning - Dungeon Escape v4.0.0**

---

## Global Shortcuts (Available Anytime)

| Shortcut | Action | Description |
|----------|--------|-------------|
| **F1** | Help | Show comprehensive help and keyboard shortcuts guide |
| **Ctrl+S** | Quick Save | Save your progress instantly (during gameplay) |
| **Ctrl+L** | Quick Load | Load your saved game (when save exists) |
| **Enter** | Submit | Submit text input / Continue dialogue |

---

## Combat Shortcuts (During Battle)

| Key | Action | Description |
|-----|--------|-------------|
| **1** | Normal Attack | Standard attack with balanced damage |
| **2** | Defend | Reduce incoming damage this turn |
| **3** | Heavy Attack | High damage attack (costs mana) |
| **4** | Quick Attack | Fast but weaker attack |
| **5** | Use Item | Use an item from your inventory |
| **6** | Run Away | Attempt to escape from combat |

**Note:** Combat shortcuts work with both regular numbers and numpad numbers.

---

## Command Shortcuts (Text Commands)

| Command | Example | Description |
|---------|---------|-------------|
| `use <item>` | `use potion` | Quick-use an item by name (works anytime) |

---

## How to Use

### Quick Save/Load
1. **During Gameplay:** Press `Ctrl+S` to save instantly
2. **From Menu:** Press `Ctrl+L` to load your saved game
3. Your progress is also auto-saved after rooms and combat

### Combat Shortcuts
1. **When fighting:** Number keys (1-6) trigger combat actions
2. **No menu navigation needed:** Just press the number
3. **Combine with mouse:** Click buttons or use keyboard - your choice!

### Help System
- **Press F1 anytime** to see the in-game help
- Shows all available shortcuts and commands
- Doesn't interrupt your gameplay

---

## Tips for Efficient Play

### Speed Running
- Use `Ctrl+S` frequently to save progress
- Combat shortcuts (1-6) speed up battles significantly
- Type `use potion` quickly during combat instead of using menus

### Accessibility
- All shortcuts are optional - mouse/text input still works
- Keyboard shortcuts make the game more accessible
- F1 help is always available if you forget a shortcut

### Best Practices
1. **Learn the combat keys (1-6)** - They save the most time
2. **Use F1 if stuck** - Quick reference without leaving game
3. **Quick save often** - Ctrl+S is faster than menu navigation
4. **Keep one hand on numpad** - For rapid combat actions

## Platform-Specific Notes

### Windows / Linux
- **Ctrl+S** = Quick Save
- **Ctrl+L** = Quick Load

### macOS
- **Cmd+S** = Quick Save (Ctrl also works)
- **Cmd+L** = Quick Load (Ctrl also works)

## Frequently Asked Questions

**Q: Can I customize keyboard shortcuts?**  
A: Not in v4.0.0, but it's planned for a future release.

**Q: Do shortcuts work on all keyboard layouts?**  
A: Yes, they're based on key codes and work with most layouts.

**Q: What if I press the wrong number in combat?**  
A: The game will execute that action - be careful! Save often with Ctrl+S.

**Q: Can I disable keyboard shortcuts?**  
A: Simply don't use them - all mouse/text input still works normally.

**Q: Do keyboard shortcuts work while typing?**  
A: F1 works anytime. Ctrl+S/L work when not actively typing. Combat keys work during combat when not waiting for text input.
